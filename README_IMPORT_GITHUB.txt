IMPORT RAPIDE DANS GITHUB

1. Dézipper ce dossier sur votre ordinateur.
2. Ouvrir votre repo GitHub "qui-est-qui-metiers".
3. Cliquer sur Add file > Upload files.
4. Glisser tout le contenu du dossier dézippé dans GitHub.
5. Commit changes.
6. Ajouter ensuite votre image ingenieur-rd.png dans assets/metiers/.
